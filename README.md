# Vue 학습노트 애플리케이션 Node.js 서버 코드

> vue 학습노트 애플리케이션의 백엔드 API를 제공하는 Node.js 코드 레포지토리입니다.

## 설치 방법

```
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```
